# microcontroller

Projects for software-hardware systems with the TI-MSPM0-G3507 microcontroller

## Contents

- LEDs: Basic practice turning lights on and off
- Sort: Input a list of numbers, sort them, get max and min, using linked list in assembly
- Keyboard: Play a MIDI keyboard on the passive buzzer
